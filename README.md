# 🍽️ Calculadora de Preços de Refeições em Python

## 📌 Sobre o projeto

Este projeto consiste em uma calculadora desenvolvida em Python para calcular o valor final de uma refeição, considerando diferentes informações como preço dos itens, impostos, descontos e pagamento realizado pelo cliente.

O objetivo deste projeto foi praticar conceitos fundamentais da programação, como entrada de dados, conversão de tipos, operações matemáticas, estruturas condicionais e organização de código.

---

## 🚀 Funcionalidades

✅ Cadastro do valor da refeição  
✅ Cálculo de impostos  
✅ Aplicação de descontos  
✅ Cálculo do valor final da compra  
✅ Entrada do valor pago pelo cliente  
✅ Cálculo do troco  
✅ Exibição do resumo da compra
